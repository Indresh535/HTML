Trouble Logging:


 
        
        You might get the following error messages at the time of logging in. Explained below is the appropriate action which has to be taken in each case:
Message 1: Enter Correct Login and/or Password
Action:
Please recheck userid / password.
Please note that Password is case sensitive.
Incase you have forgotten your password, please submit a request for reset of password through the Forgot Password Link. We will authenticate you on the basis of the details submitted.

Message 2: User has not opted for Admission Portal
Action:
Download the application form, register for Admission Poratl and submit it at your College.

Message 3: User pending for activation, kindly contact College for activation
Action:
Please contact College and get you User Id activated, to use Admission Portal facility.

Message 4: User Status has become Dormant. Please contact College
Action:
Please contact College and get you User Id activated, to use Admission Portal facility.

Message 5: User status is locked, kindly contact College
Action:
Please contact College and get you User Id activated, to use Admission Portal facility.

Message 6: User Id is Deleted
Action:
Please contact College and get you User Id activated, to use Admission Portal facility.

Message 7: User Validity Expired
Action:
Please contact College and get you User Id activated, to use Admission Portal facility.

Message 8: User already logged on. Try logging in after closing the current session
Action:
Please Logoff your existing session and login into a new session.

Message 9: You are logging in another Session.Please login in another Browser
Action:
Please Logoff your existing session and login into a new session.

Message 10: Your request could not be processed Try after some time
Action:
This happens if there is a communication problem. Please try logging in after sometime.
